## What is PyTECGg?

PyTECGg is a fast, lightweight Python package to **reconstruct and calibrate** TEC from **GNSS** measurements.

## Main features

- Supports RINEX v2-3-4
- High-performance Rust core
- Linear combinations (GFLC, IFLC, MW)
- Cycle-slip / loss-of-lock detection
- Satellite coordinates + IPP computation
- TEC calibration (sTEC + vTEC)
